# Arch Linux Me Config

This repository contains **old samples** for some config files and shell scripts from the Arch Linux operating system.

---------------------------------------------------------------

> [!WARNING]
> Uses only as a reference for creating your own configs and scripts!

---------------------------------------------------------------

**That might be useful in setup by examples, e.g**:

* ext-ip - Script for get public ipv4 address.
* ipv4conv6to4 - Here public ipv4 address convert ipv6 address for 6to4.
* Configs and scripts for stunnel, inetd.
* Etc...
